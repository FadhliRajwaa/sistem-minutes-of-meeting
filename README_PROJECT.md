# Sistem Minutes of Meeting

Aplikasi web untuk mengelola notulensi rapat, peserta, dan absensi.
Dikembangkan menggunakan CodeIgniter 4, Bootstrap 5, dan terintegrasi dengan database Aiven MySQL.

## Fitur
- Dashboard Statistik
- Manajemen Meeting
- Manajemen Peserta (Scan Barcode)
- Diskusi & Notulensi
- Absensi Realtime
- Export PDF
- Login/Register Authentication

## Deployment
Aplikasi ini dikonfigurasi untuk deployment di Vercel dengan database MySQL di Aiven.

### Requirements
- PHP 8.1+
- MySQL
- Composer

### Setup
1. Clone repository
2. `composer install`
3. Setup Environment Variables (lihat `.env.example`)
4. `php spark migrate`
